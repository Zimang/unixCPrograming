# Systems Programming in Unix/Linux K.C.Wang
## 介绍
### 系统编程的作用

1. 提供系统软件和高级编程的广泛知识，接触操作系统内核接口能利用系统资源应用层软件
2. 提供足够的再学习背景

## 该书内容

### 编程练习

包括程序开发的详细过程（assembler, compiler, linker, link library, executable file contents, program execution image, function call conventions, parameter passing schemes, local variables, stack frames, link C program
with assembly code, program termination and exception handling.）

包括如何使用Makefiles管理大型编程项目，GDB来debug
